## 📖 Overview
This is a zero-config command-line interface tool that scans Python and JavaScript codebases to construct a call graph and identify Zombie Functions that are defined but never called. It addresses the inefficiency of technical debt by eliminating unused code without the complex setup or heavy overhead of alternative analysis frameworks. The tool generates a Bill of Mortality report to pinpoint unused functions and imports, directly aiding in reducing bloat and context window costs. This solution is built for developers and technical teams who need immediate, frictionless code hygiene verification to streamline their refactoring process.
